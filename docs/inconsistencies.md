**Suggested issues:**
1. Convert `SavedMessages.Web` to Blazor WASM (or explicitly decide to keep it as Blazor Server and update all docs).

## INC-09 · MinIO credentials hardcoded in `Program.cs` (security)

**Affected docs:** No related issue; ARCHITECTURE.md §6 says "Azure Key Vault holds all secrets"  
**Reality:** `SavedMessages.ApiService/Program.cs` hardcodes `"minioadmin"/"minioadmin"` as the S3 access key and secret:
```csharp
return new AmazonS3Client("minioadmin", "minioadmin", s3Config);
```
No secret manager or environment variable is used.

**Suggested issue:** Replace hardcoded MinIO credentials with configuration-backed values (`IConfiguration`), inject them via Aspire resource environment variables in development, and document the required secret names.
