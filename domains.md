# Alternative Name & Domain Options

## Inspiration: Windows 95/98 Briefcase

The original **My Briefcase** in Windows 95–98 was a folder you'd drag files into, carry to another machine on a floppy, and sync back. The metaphor maps perfectly: a personal container you carry between your own devices. "Brief" also carries a double meaning — *brief* as in short, concise messages.
